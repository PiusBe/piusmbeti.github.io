HR Data Analysis

  • Tech Stack: SQL, Power BI
  • Github URL: Project Link

This data analysis project examines HR data from 2000 to 2020, comprising 22,000 rows cleaned using MySQL. Its main aim is to gain valuable insights into various HR aspects, informing decision-making and strategies. The analysis addresses specific questions about employee demographics, gender distribution, race, age, work locations, tenure, turnover rates, and geographic distribution.

Findings show a gender imbalance with more male employees, White as the dominant race, and a majority of employees aged 25-34. Headquarters is the preferred work location, while the Marketing department has the highest turnover rate. The report also identifies the state of Ohio as a significant source of employees. Despite some limitations, the analysis offers valuable information for HR decision-making.
